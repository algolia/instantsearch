The library of UI widgets to build the best instant-search experience with Algolia's Hosted Search API.

There’s a widget for that

Pick from an extensive library of widgets to power your search
and build a state of the art instant search experience.

More than widgets

You code becomes easy to write and maintain as you won't need to build all the glue code between widgets to track the state of your facets, decode URL parameters, etc. Everything is packaged in the lib.

Infinite possibilities

Every website is different. The modular design of instantsearch.js allows you to configure, rearrange and combine the widgets to build your own experience.

Bring your own widget

Sometimes you want to deviate from the standard path and build a custom feature. You can do that by forking a widget or coding your own and integrating into the open source project.
